### Install Python Libraries
You can run this python script from terminal but you will need the following libraries: 
  - PyQt4
  - Numpy
  - Matplotlib
  - Scipy 
  - OpenCV
  - NdImage
  - Imutils 

### Running the Program

From Terminal: python Gui2A.py
